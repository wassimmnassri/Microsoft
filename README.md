# Microsoft 365 Developer Project

## Description
This project is created as part of the **Microsoft 365 Developer Program**.  
It focuses on learning and experimenting with **Microsoft 365**, **Microsoft Graph API**, and **SharePoint Online**.

The goal is to understand how to build cloud-based applications that interact with Microsoft 365 services in a secure and modern way.

## Technologies
- Microsoft 365
- Microsoft Graph API
- SharePoint Online
- Azure AD (Entra ID)
- REST APIs
- OAuth 2.0 Authentication

## Learning Objectives
- Access Microsoft 365 data using Microsoft Graph
- Authenticate applications with Azure AD
- Work with SharePoint sites, lists, and users
- Understand Microsoft 365 developer tools and concepts
